# Optional Directus Setup

Directus run karne ke liye Docker required hai.

```bash
cd directus-optional
docker compose up -d
```

Open:

```text
http://localhost:8055
```

Note: Ye optional backend/admin studio hai. Main Niharo Node app apne free file-based backend ke saath already chalti hai. Directus ko final production architecture me tab use karein jab aap SQL database, roles/permissions aur hosted deployment decide kar lein.
